simple setup for i3wm and some tools based on stock endeavourOS config for i3 which can be found [here](https://github.com/endeavouros-team/endeavouros-i3wm-setup).

files can simply be dropped into the home folder in an eos i3 setup, but you need to install the Avant Garde Regular and AvantGarde Medium fonts and apply the GTK theme (you can use [nwg-look](https://github.com/nwg-piotr/nwg-look) which also comes preinstalled in eos)
